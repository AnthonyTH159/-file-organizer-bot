# File Organizer Bot – Python Automation Script

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Tired of a messy Downloads folder?** This Python script automatically organizes files into categorized subfolders (Images, Documents, Videos, etc.) — saving you hours!

### Features
- Categorizes files by extension (expandable dictionary)
- Creates destination folders automatically
- Handles duplicate names by adding numbers
- Dry-run mode to preview changes
- Command-line arguments for easy use
